# Project: product search by image and text
## Table of content

## Introduction
This project includes:
- **Application (FastAPI)**: user can upsert a new product with metadata, search by product image or search by text.
- **ML Model**: CLIP model to generate embedding vectors
- **Vector Database**: Pinecone for efficient vector search.
- **Observability Stack**: a complete monitoring solution with Prometheus, Grafana, Jaeger, and the ELK stack.
- **CI/CD pipeline**: A fully automated `Test -> Build -> Deploy` pipeline using Jenkins with PyTest.
